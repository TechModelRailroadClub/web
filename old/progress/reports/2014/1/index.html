This page describes work accomplished on the new TMRC layout during the month of January 2014, as taken from the descriptions posted to the club mailing list.
January 11

An attempt at creating a transition rail from code 70 down to code 55, by crushing the rail. This doesn't look pretty, but there is hope for the technique.
Paper mill ties pre-stained with Rust-Oleum "Sunbleached".
Ties now stained with dilute Walnut, with the idea of producing a grey-brown tone that looks like old weathered wood.
More stained ballast has been mixed--now drying on the radiator (and perfuming the clubroom).
Close-up of ballasted ties with a few weeds added, from the "Leder Mix" jar.


January 22

The diagram for the east end of F-Yard has been reprinted with a couple of changes.
A new turnout at the paper mill.
This shows the size difference between code 83 on the main line, and code 55 on the mill tracks.


January 29

Upper level trackwork is taking shape.
Andy is considering roads and placement of buildings in East Berkmannvile.
